# React App Example

A minimal React application with Howth dev server.

## Quick Start

```bash
# Install dependencies
howth install

# Start dev server with HMR
howth dev src/main.tsx --port 3000 --open
```

## Files

- `src/main.tsx` - React entry point
- `src/App.tsx` - Main component with state
- `src/Counter.tsx` - Counter component demonstrating hooks
- `package.json` - Dependencies (react, react-dom)

## Features Demonstrated

- React with TypeScript
- Hot Module Replacement (HMR)
- React Fast Refresh (state preserved on edit)
- JSX/TSX support
- ES module imports
