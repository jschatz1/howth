# HTTP Server Example

A simple HTTP server using Howth's native runtime.

## Quick Start

```bash
howth run server.js
```

Then visit http://localhost:3000

## Files

- `server.js` - Basic HTTP server with routing
- `server-json.js` - JSON API server example

## Features Demonstrated

- Creating an HTTP server with `http.createServer()`
- Request routing
- JSON responses
- URL parsing
