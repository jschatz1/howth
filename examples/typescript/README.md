# TypeScript Example

Run TypeScript directly without a build step.

## Quick Start

```bash
howth run main.ts
```

## Files

- `main.ts` - Entry point with type annotations
- `utils.ts` - Helper functions with interfaces
- `types.ts` - Shared type definitions

## Features Demonstrated

- Direct TypeScript execution (no tsc needed)
- Type annotations and interfaces
- ES modules with TypeScript
- Importing .ts files
