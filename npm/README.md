# howth

A fast JavaScript runtime.

## Installation

```bash
npm install -g howth
```

## Usage

```bash
howth run script.js
howth test
howth --help
```

## Manual Installation

If npm installation fails, you can install manually:

```bash
curl -fsSL https://raw.githubusercontent.com/jschatz1/howth/main/install.sh | sh
```
